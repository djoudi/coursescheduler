Course Schedule Creator

Overview

This web based software is designed to simplify making changes to dates in course calendars. It displays the calendar in a table of rows with each row representing 1 week. The number of weeks displayed is a chosen variable, though a typical Fall or Spring semester is 16 weeks, and Summer semester 8 weeks. Unlimited lessons and assignments with due dates may be added to columns in a given week.

Dates entered for Assignments Due create a stored factor based on the start date of the course. This means all assignments remain the same length of time from the start date no matter what the start date is. Thus changing the start date changes all dates in the calendar.

Changing Semester between Fall & Spring inserts Spring & Thanksgiving breaks in the proper week and alters all Lesson and Assignment dates accordingly.

Different color schemes can be chosen which alter the styling of the final display.

Course Schedule Creator - the starting page

There are three options: Create New Course Schedule, Edit Existing Course, Save Schedule of Existing Course as an HTML Page.

Create New Course Schedule
Make your selections -all of this can be edited later if necessary.

Click Create New Course.
This will save entered choices to a database and then redirect to the editing page where Lessons and Assignments are added. The next time you load the start page your new course will be listed in the drop down lists on this page.

Edit Existing Course
Select a course from the list. This will direct you to the editing page for that course.

Save Schedule of Existing Course as an HTML File
Select a course from the list. This will create an HTML file on the server of the chosen schedule and return a link to that file.

Course Schedule Editing Page

  • This is where all lessons and assignments can be added, deleted, or edited. These are saved as they are made.
  • All choices made in the initial set up of the course can be changed. Click Change Course Settings to save.
  • It is also possible to save all the data from one course to a New Name. You might do this if you wanted a calendar that was just a little different for some reason. Enter the New Course name and click Save.
  • There is an option to add Course Notes. This text will be displayed in a small font size just beneath the course title.
  • Course schedules can be shared with other users who have an account. In upper right select user ID from drop down list and click Add User. All users have equal rights.

Using the Finished Schedule

  • Selecting Display Schedule from the top right of Editing page will display the finished styled calendar.
  • From your browser click File > Save As - Change the name to what you want. Remove the .php extension and keep the .html extension.
  • This file can now be uploaded to an Angel course and linked to the Calendar Tab. It can be opened in Word and saved as a word document. It can be opened in Dreamweaver and modified further if necessary.
Or...
  • From the Options page Save Existing Schedule as an HTML file.
  • This will save an HTML file from data of the selected course and give you a URL to that file.
  • This file can be downloaded and altered as above.
